## Preset Applicatie

### CSS Variables Export

Elk preset genereert de volgende CSS met light en dark mode:

```css
:root {
  /* Main Colors */
  --color-dark: [preset.dark];
  --color-light: [preset.light];
  --color-mid-gray: [preset.mid-gray];
  --color-light-gray: [preset.light-gray];

  /* Accent Colors */
  --color-accent-primary: [preset.accent-primary];
  --color-accent-secondary: [preset.accent-secondary];
  --color-accent-tertiary: [preset.accent-tertiary];

  /* Semantic Colors */
  --color-success: [preset.success];
  --color-warning: [preset.warning];
  --color-error: [preset.error];
  --color-info: [preset.info];

  /* Typography */
  --font-heading: [preset.font-heading], [preset.heading-fallback];
  --font-body: [preset.font-body], [preset.body-fallback];
  --font-mono: [preset.font-mono], [preset.mono-fallback];

  /* Theme Mode Variables (Light - default) */
  --background: [preset.light];
  --foreground: [preset.dark];
  --card: [preset.light-gray];
  --card-foreground: [preset.dark];
  --border: [preset.mid-gray];
  --input: [preset.light-gray];
  --ring: [preset.accent-primary];
  --accent: [preset.accent-primary];
  --accent-foreground: [preset.light];
}

.dark {
  /* Theme Mode Variables (Dark) */
  --background: [preset.dark-background];
  --foreground: [preset.dark-foreground];
  --card: [preset.dark-card];
  --card-foreground: [preset.dark-foreground];
  --border: [preset.dark-border];
  --input: [preset.dark-background];
  --ring: [preset.dark-accent-primary];
  --accent: [preset.dark-accent-primary];
  --accent-foreground: [preset.dark-background];

  /* Adjusted accent colors for dark mode */
  --color-accent-primary: [preset.dark-accent-primary];
  --color-accent-secondary: [preset.dark-accent-secondary];
  --color-accent-tertiary: [preset.dark-accent-tertiary];
}
```
